# 员工管理系统 java968

## 项目概述

员工管理系统是一个基于Web的软件应用，采用前后端分离的技术架构。系统旨在为各类企业提供便捷高效的员工管理解决方案。

## 技术栈

### 系统架构

- B/S架构
- MVC模式

### 系统环境

- Windows

### 开发环境

- IDEA
- JDK 1.8
- Maven
- Mysql
- Node.js

### 技术组件

- Java
- Mysql
- Spring Boot
- MyBatis
- Vue
- 文件上传与Excel生成下载
- Element UI

## 功能模块

系统设计了以下功能模块：

- 部门管理
- 职工管理
- 考勤管理
- 奖惩管理
- 公告管理
- 合同管理

## 系统角色

系统定义了以下三种角色，每种角色拥有不同的菜单权限：

- 管理员
- 人事经理
- 职工

## 前端实现

前端使用Vue框架，并已打包为静态资源文件（dist）。用户可以直接启动后端服务访问前端页面，也可以自行通过npm安装依赖并启动Vue。

## 部署步骤

1. 准备后端运行环境，包括Java、Mysql等。
2. 部署Spring Boot后端服务。
3. 将打包好的Vue前端资源放置到服务器指定目录。
4. 启动后端服务，即可访问系统。

## 目录结构

```
- resources（或webapp）
  - admin（未打包的Vue代码）
  - ...
```

## 核心亮点

- 系统基于成熟的Spring Boot和Vue技术栈，确保了稳定性和扩展性。
- 前后端分离，易于维护和升级。
- 清晰的角色定义和权限控制，保证系统的安全性和管理效率。
- 实现了文件上传和Excel文件的生成下载，提高了数据处理能力。
